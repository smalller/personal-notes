## 🌱 常规版使用

```bash
# 克隆项目
git clone https://github.com/vue-admin-beautiful/vue-admin-beautiful-pro.git
# 进入项目目录
cd vue-admin-beautiful-pro
# 安装依赖
npm i
# 本地开发 启动项目
npm run serve
```
